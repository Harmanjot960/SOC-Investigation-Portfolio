# PCAP Evidence

The original packet capture used for this investigation was analyzed using:

- Wireshark
- Suricata IDS

The PCAP was obtained from:

Malware Traffic Analysis — 2025-01-22 Traffic Analysis Exercise

https://www.malware-traffic-analysis.net/

The packet capture contained approximately 39,000 packets and was analyzed to identify:

- Malware download activity
- PowerShell payload delivery
- Command-and-control communication
- Suspicious TLS connections
- Indicators of compromise

Due to repository size limitations, the original PCAP file is not included.

Generated investigation artifacts from the PCAP analysis are included in:

- Screenshots/
- Evidence/
- Suricata/
- Incident-Report/
- MITRE-ATT&CK/
